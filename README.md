# STAR AIDE 🚀
**The Ultimate NDK Development Environment for Android**

STAR AIDE is a high-performance development tool designed for Android developers to manage, edit, and build C/C++ projects (NDK) directly on their mobile devices.

## ✨ Key Features
*   **Project Management:** Create fresh NDK projects or import existing directories from storage easily.
*   **Integrated NDK Setup:** Built-in downloader and configurator to set up your NDK environment in seconds.
*   **Modern UI/UX:** Professional Material Design with smooth animations and responsive layouts.
*   **Optimized for Android 11+:** Full support for "All Files Access" (Scoped Storage) for efficient file management.
*   **Customizable Editor:** Adjust font sizes and preferences to create your perfect coding environment.
*   **Performance Focused:** Lightweight and built to handle native build processes without lag.

## 📥 How to Install
1. Download the latest **APK** from the [Releases](https://github.com/botaltier2v-coder/STAR-AIDE/releases) section.
2. Install the app on your Android device.
3. Grant **"All Files Access"** permission to allow the app to manage your source code.
4. Install or configure the **NDK** via the internal settings menu.
5. Start building your native C++ applications!
