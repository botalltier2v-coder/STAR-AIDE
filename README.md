
# STAR AIDE 🚀
**The Ultimate NDK Development Environment for Android**

STAR AIDE adalah aplikasi mobile yang dirancang untuk memudahkan developer Android dalam mengelola, mengedit, dan melakukan build proyek C/C++ (NDK) langsung dari smartphone.

## ✨ Fitur Unggulan
- **Full NDK Support:** Terintegrasi dengan peralatan build native.
- **Modern Material UI:** Antarmuka bersih dengan dukungan Dark Mode.
- **Advanced File Manager:** Optimal untuk Android 11+ (Scoped Storage).
- **Lightweight & Fast:** Dioptimalkan untuk performa tinggi tanpa lag.
- **Customizable Editor:** Atur kenyamanan coding Anda sendiri.

## 📥 Instalasi
1. Download APK terbaru dari tab [Releases].
2. Berikan izin "All Files Access" untuk mengelola project.
3. Install NDK melalui menu di dalam aplikasi.
