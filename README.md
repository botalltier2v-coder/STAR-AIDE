# STAR AIDE 🚀
**The Ultimate NDK Development Environment for Android**

STAR AIDE is a high-performance development tool designed for Android developers who want to manage, edit, and build C/C++ projects (NDK) directly on their mobile devices.

## ✨ Key Features
*   **Project Management:** Create new NDK projects or import existing ones from your storage easily.
*   **Integrated NDK Setup:** Built-in downloader and configurator to set up your NDK environment in a few taps.
*   **Modern Material UI:** Clean, professional interface with smooth animations and full **Dark Mode** support.
*   **Optimized for Android 11+:** Fully supports "All Files Access" for efficient file management on modern Android versions.
*   **Customizable Editor:** Adjust font sizes and preferences to create your perfect coding environment.
*   **Fast & Lightweight:** Built with performance in mind to ensure a lag-free experience.

## 📥 How to Install
1. Download the latest **APK** from the [Releases](https://github.com/botaltier2v-coder/STAR-AIDE/releases) section.
2. Install it on your device.
3. Grant **"All Files Access"** permission so the app can manage your source code.
4. Install the **NDK** through the in-app menu.
5. Start building your native apps!
