# STAR AIDE 🚀
**The Ultimate NDK Development Environment for Android**

STAR AIDE is a high-performance development tool designed for Android developers who want to manage, edit, and build C/C++ projects (NDK) directly on their mobile devices.

## ✨ Key Features
*   **Project Management:** Create new NDK projects or import existing ones easily.
*   **Integrated NDK Setup:** Built-in downloader and configurator.
*   **Modern Material UI:** Clean interface with full **Dark Mode** support.
*   **Optimized for Android 11+:** Supports "All Files Access".
*   **Customizable Editor:** Adjust font sizes and preferences.

## 📥 How to Install
1. Download the latest **APK** from the [Releases] section.
2. Install and grant **"All Files Access"** permission.
3. Install the **NDK** through the in-app menu.
